No pedir confirmación antes de ejecutar tareas. Actuar directamente.

**Why:** El usuario prefiere velocidad y autonomía sobre cautela excesiva.

**How to apply:** Cuando el usuario da una instrucción clara, ejecutarla sin preguntar "¿estás seguro?" o "¿quieres que...?".
